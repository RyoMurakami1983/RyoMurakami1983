## 🧠 My Philosophy & WILL-ACTION Declaration

### My WILL（志）

**「余白を守り誰もが挑戦できる基礎と型を作り、自分がいなくても成長が連鎖する組織を育てる」**

#### Why? ——背景と信念

**原体験：高校時代の「退屈」── 余白の再発見**  
高校時代の「何もしない退屈」は、当時は人生最低の体験だった。だが今振り返れば、あれこそが**余白**だった。何もなかったからこそ、**挑戦し続けるほうが面白い**という原点が刻まれた。

**転機：2017年の余白**

担当事業の譲渡で、思いがけず余白が生まれた。その隙間で『7つの習慣』に出会い、自分のミッションを考え始めた。**余白がなければ、基礎に触れることすらできなかった。**

**2つの発見の統合**

生成AIとの出会いがもたらした **「知識の増幅可能性」** と、空手指導で培った **「暗黙知を相手に合わせて言語化し、対話する技術」** ——この2つを組み合わせれば、誰もが基礎から応用を生み出せる環境を作れると確信しています。

**「自分がいなくても世界は回る」仕組みを作った先に、本当に自分にしかできることと、チーム全体が自律成長する世界がある** と信じています。

#### How? ——具体的な行動

**日常では**、自身の技術を原理原則レベルまで基礎を掘り下げます。

**非日常では**、部門横断の勉強会やコミュニティを主催し、教え･対話しながら専門性を高めます。

**そして**、生成AIによる業務自動化で生まれた時間を、「再利用可能な型とベストプラクティスの確立」「子どもへの武道・空手指導」「チームが自走する仕組みづくり」「省人化の仕組みづくり」に注ぎます。

個人の速さを保ちながら、チーム全体で再現できる構造を設計し、挑戦を継続できる最低限の安定の上で、常に新しいことに取り組んでいきます。

## 🎯 VISION・MISSION・VALUES

### Vision（未来・結果）

**基礎から応用が生まれ続ける、自律成長する組織と人々**

- 私がいなくても、誰かが新しい挑戦を始められる
- 型があるから速く動け、基礎があるから応用が尽きない
- 子どもも、チームも、コミュニティも、それぞれが成長を楽しんでいる

### Mission（行動）

**揺るがない基礎と再利用可能な型をコツコツ作り、教え、仕組みに埋め込む**

1. **新しい技術を原理原則レベルまで掘る**
   - プログラミング100日チャレンジで基礎を深掘り

2. **暗黙知を相手に合わせて言語化し、伝える**
   - 部門横断の勉強会主催で専門性を高める
   - ナレッジベースの整備と形式知化

3. **個人の速さを保ちながら、チームで再現できる仕組みを設計する**
   - 設計パターン、ベストプラクティスの確立
   - チームが自走する構造づくり

4. **生成AIと自動化で時間を生み、「自分にしかできないこと」に挑む**
   - 業務自動化で個人の時間創出
   - 子どもへの武道・空手指導
   - 省人化の仕組みづくり

### Values（大切にしていること）

1. **温故知新** → 過去の知恵と新しい技術を繋ぎ、時間軸を超えた価値を創造
2. **継続は力なり** → 退屈しない挑戦をコツコツ積み上げ、個人と組織の複利成長へ
3. **基礎と型の追求** → 最小形式で最大可能性を生む設計思想。すべての応用は基礎から
4. **成長の複利** → 自分も周囲も同時に成長する構造をつくり､教える喜び、学ぶ喜び
5. **ニュートラルな視点** → 偏らず、誰もが使える普遍性を保つ。個人知を形式知に
6. **余白の設計** → 学びと変化が生まれる隙間を、構造的に守る
